БАНКОВСКАЯ СИСТЕМА
Приложение представляет собой некую условную банковскую систему, куда можно добавлять клиентов (физических или юридических лиц), открывать для них счета до востребования, депозиты и кредиты.
Также есть возможность отправлять деньги между счетами до востребовния, рассчитывать сумму на депозите ожидаемую к определенной дате и определять сумму ежемесячных платежей по кредиту.
Приложение состоит из ТРЕХ частей - WebApi-сервера c базой данных, WPF-приложения с применением паттерна MVP и ASP.NET Core MVC приложения. 

Что нужно сделать для запуска приложения:

1. Открыть в Visual Studio проект BankWebApi.
2. Средства > Диспечер пакетов NuGet> Консоль диспечера пакетов.

3. Запустить проект BankWebApi в Visual Studio (при первом запуске база данных наполнится автоматически).

4. Открыть в Visual Studio проект ASP_BankWebApi.
5. Снова: Средства > Диспечер пакетов NuGet> Консоль диспечера пакетов.
6. В консоли написать "update-database".
7. Запустить проект ASP_BankWebApi в Visual Studio.

8.Открыть в Visual Studio проект BankWPFApi.
9. Запустить проект ASP_BankWebApi в Visual Studio.

10. Приложение готово к использованию.
