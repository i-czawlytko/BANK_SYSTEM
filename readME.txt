1. Открыть в Visual Studio проект BankWebApi.
2. Средства > Диспечер пакетов NuGet> Консоль диспечера пакетов.
3. В консоли написать "update-database".
4. Запустить проект BankWebApi в Visual Studio (при первом запуске база данных наполнится автоматически).

5. Открыть в Visual Studio проект ASP_BankWebApi.
6. Снова: Средства > Диспечер пакетов NuGet> Консоль диспечера пакетов.
7. В консоли написать "update-database".
8. Запустить проект ASP_BankWebApi в Visual Studio.

9.Открыть в Visual Studio проект BankWPFApi.
10. Запустить проект ASP_BankWebApi в Visual Studio.

11. Приложение готово к использованию.
